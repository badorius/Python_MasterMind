Mié, 09/21/2011 - 10:26 — badorius

Para ver que Dblinks hay en una BD Oracle, lo podemos consultar de la siguiente forma:  

`select * from DBA_DB_LINKS;  

select * from ALL_DB_LINKS;  

select * from USER_DB_LINKS;`
