Lun, 06/21/2010 - 15:35 — badorius

Para lanzar un script .py desde WLST:


Llamamos a la consola WLST:


 `java weblogic.WLST`


Connectamos:


 `connect ('usuario','password','t3://servidor:7001')`


Ejecutamos el script:


 `execfile ('/home/weblogic/WLST/deployList.py')`
