[Inicio](/) ## Linux - Serial Number con dmidecode

 

Vie, 08/30/2013 - 12:17 — badorius

Para ver el serial number de una máquina:  

 `dmidecode -s system-serial-number`


También puedes ver el resto de información con:  

 `dmidecode | less`
